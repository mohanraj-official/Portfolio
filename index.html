Credit Card Fraud Detection

Duration: Jan 2024 – Feb 2025

Organization: Entri Elevate

Overview: Built ML models to detect fraudulent transactions from credit card data.

Key Steps:

  • Data Cleaning: Handled missing values and applied sampling to balance the dataset.
  • EDA: Used bar charts, box plots to analyze features like transaction amount and fraud patterns.
  • Feature Selection: Removed less relevant attributes and retained key fraud indicators.
  • Model Building: Trained Logistic Regression, Random Forest, and XGBoost classifiers.
  • Evaluation: Compared models using Precision, Recall, and AUC-ROC metrics.
  • Optimization: Tuned hyperparameters for better performance.
  • Deployment (Optional): Designed for real-time fraud detection integration.

Tools: Python, Pandas, Matplotlib, Sklearn, XGBoost

Heart Attack Trends: Youth vs Adults (China)

Duration: Dec 2024 – Jan 2025

Organization: Entri Elevate

Overview: Analyzed 270,000 health records to compare heart attack risks between youth (under 30) and adults (over 30) in China.

Key Steps:

  • Data Analysis: Used EDA to uncover patterns based on age, health metrics, lifestyle, and habits.
  • Risk Indicators: Identified key features such as blood pressure, cholesterol, BMI, and physical activity.
  • Model Training: Built ML models to predict heart attack risk based on age-specific features.
  • Insights: Found both age groups at risk, but with different influencing factors.

Tools: Python, Pandas, Sklearn, Matplotlib, Seaborn

Python - Marketing Campaign Performance Insights

Duration: Oct 2024

Organization: Entri Elevate

In this project, I analyzed the effectiveness of a marketing campaign by examining customer responses, sales data, and various metrics. I began by collecting and cleaning the dataset, which included customer demographics and campaign details.

Through exploratory data analysis (EDA), I identified patterns in how different customer groups responded. I segmented customers by age, income, and location to pinpoint responsive audiences.

Visualizations such as bar and pie charts were created to effectively communicate findings and strategies.

This project improved my skills in data analysis, customer segmentation, machine learning, and data visualization.

Library Management and Attendance System

Duration: Jan 2024 – Mar 2024

Organization: Global Institute of Engineering and Technology

The Library Attendance Management System is an IoT-based solution designed to automate student attendance using facial recognition technology. It uses a webcam or camera module to capture student faces upon entry and exit, ensuring accurate and efficient attendance tracking.

The system was built using Python and OpenCV for facial recognition, with a MySQL database for storing attendance records. I developed the user interface (UI) using HTML, CSS, and JavaScript, which connects with the backend via a XAMPP server.

The facial recognition module automatically matches captured images with stored student profiles to log attendance. This significantly reduced manual effort and errors while offering a smooth experience for both students and library staff.

Skills: Facial Recognition · XAMPP · PHP · OpenCV · Internet of Things (IoT)
